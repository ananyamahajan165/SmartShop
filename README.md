# 🛒 Smart Shop Management System

The **Smart Shop Management System** is a Java-based application that simulates a simple digital shop.  
It allows users to view products, add items to a cart, generate bills, and manage basic shop operations.

This project is created for **academic learning purposes** and demonstrates the use of **core Java concepts** in a real-world scenario.


## 🚀 Features

- 📦 Product listing with prices
- ➕ Add items to cart
- 🧾 Automatic bill generation
- 💰 Total cost calculation
- 🔁 User-friendly menu-driven interface
- 🛠️ Modular and readable Java code
  
## 🧠 Concepts Used

- Object-Oriented Programming (OOPS)
  - Classes & Objects  
  - Encapsulation  
  - Methods & Constructors
- Java Collections (ArrayList / HashMap)
- Conditional Statements & Loops
- Basic Input/Output handling
- Clean code structure


## 🖥️ Tech Stack

- **Language:** Java  
- **IDE:** VS Code / IntelliJ / Eclipse  
- **Platform:** Console-based application
